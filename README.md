## Repository_Overview

The codebase is organized chronologically, reflecting the weekly curriculum and the increasing complexity of the concepts covered.

### Curriculum_Progression

*   **Week 01: Course Overview**
      [![Dynamic_Web_F25_Graduate](https://img.shields.io/badge/Dynamic_Web_F25_Graduate-000000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/kadee80/dynamic_web_f25_graduate)

*   **Week 02-05: Foundations**
    *   Introduction to HTML5, CSS3, and JavaScript (ES6+).
    *   Responsive design principles and DOM manipulation.
    *   Initial exploration of component-based thinking.

*   **Week 06: React Fundamentals**
    *   Introduction to the React library.
    *   State management with `useState` and side effects with `useEffect`.
    *   Project: Mini Apps (Memory Game, Image Search).

*   **Week 07-09: Midterm Project (ResumeAI)**
    *   Development of a full-stack application using Next.js.
    *   Integration with Supabase for authentication and database management.
    *   Implementation of AI features using OpenAI API.
    *   Evolution from static prototype (v1) to functional application (v2).

*   **Week 10-12: Advanced State Management**
    *   Deep dive into React Hooks (`useReducer`, `useContext`).
    *   Project: Advanced Todo Application with complex state logic.

*   **Week 13-14: API Integration & UI Frameworks**
    *   Working with external APIs (Harvard Art Museums).
    *   Advanced styling with Tailwind CSS and component libraries.
    *   Project: Art Collection Explorer.

*   **Week 15: Final Project (DAILY_FILM)**
    *   Culmination of the semester's learning.
    *   A production-grade movie discovery platform built with React 19, Vite, and TMDb API.
    *   Features include custom caching strategies, bionic reading, and industrial design aesthetics.
